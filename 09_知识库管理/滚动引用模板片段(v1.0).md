# 🔄 滚动引用模板片段

## 用途

将此 DataviewJS 代码块放在笔记**最顶部**（标题前面），打开笔记时会自动：

1. 读取当前笔记的 `tags`
2. 在 [[09_知识库管理/引用数据库|引用数据库]] 中匹配对应的类别
3. 随机选一条语录显示在 callout 框中
4. 每次打开笔记显示不同的语录（滚动效果）

## 支持的标签 → 语录类别映射

| 笔记标签 | 语录类别 | 示例效果 |
|----------|---------|---------|
| `交易日志`、`交易`、`复盘` | 交易日志 | 交易纪律 |
| `学习`、`学习笔记` | 学习 | 学习原则 |
| `专业`、`事业`、`项目` | 专业 | 工作规范 |
| `认知`、`思维` | 认知 | 思维模型 |
| `日记`、`生活` | 日记 | 人生信条 |
| `写作`、`笔记` | 写作 | 写作原则 |

> [!warning] 前提条件
> 必须安装并启用 **Dataview 插件**，否则代码块不会渲染为语录。

## 代码片段（直接复制）

````markdown
```dataviewjs
// ==========================================
// 滚动引用 - 根据标签随机显示语录
// 数据源: [[09_知识库管理/引用数据库]]
// ==========================================

// 1. 从引用数据库中提取所有语录
const quotes = dv.pages('"09_知识库管理/引用数据库"')
  .flatMap(p => {
    const results = [];
    const items = p.file.lists;
    for (const item of items) {
      if (item.类别 && item.语录) {
        results.push({
          category: item.类别,
          text: item.语录,
          author: (item.作者 && item.作者.trim() !== '') ? item.作者 : null
        });
      }
    }
    return results;
  });

// 2. 获取当前笔记的标签，匹配语录类别
const fileTags = dv.current().tags || [];

// 标签 → 类别 映射关系
const tagToCategory = {
  "交易日志": "交易日志",
  "交易": "交易日志",
  "复盘": "交易日志",
  "月总结": "交易日志",
  "周总结": "交易日志",
  "学习": "学习",
  "学习笔记": "学习",
  "阅读": "学习",
  "专业": "专业",
  "事业": "专业",
  "工作": "专业",
  "项目": "专业",
  "面试": "专业",
  "认知": "认知",
  "思维": "认知",
  "思维模型": "认知",
  "日记": "日记",
  "生活": "日记",
  "健康": "日记",
  "写作": "写作",
  "笔记": "写作",
  "个人成长": "学习"
};

// 3. 查找匹配的语录
let matchedCategory = null;
for (const tag of fileTags) {
  if (tagToCategory[tag]) {
    matchedCategory = tagToCategory[tag];
    break;
  }
}

// 如果标签没匹配到，尝试用路径中的关键词匹配
if (!matchedCategory) {
  const path = dv.current().file.path;
  if (path.contains("交易") || path.contains("执行") || path.contains("财富")) {
    matchedCategory = "交易日志";
  } else if (path.contains("学习") || path.contains("成长")) {
    matchedCategory = "学习";
  } else if (path.contains("专业") || path.contains("事业") || path.contains("项目")) {
    matchedCategory = "专业";
  } else if (path.contains("认知") || path.contains("思维")) {
    matchedCategory = "认知";
  } else if (path.contains("生活") || path.contains("健康") || path.contains("日记")) {
    matchedCategory = "日记";
  } else if (path.contains("写作")) {
    matchedCategory = "写作";
  }
}

const matchedQuotes = matchedCategory
  ? quotes.filter(q => q.category === matchedCategory)
  : [];

// 4. 随机选一条显示
if (matchedQuotes.length > 0) {
  const q = matchedQuotes[Math.floor(Math.random() * matchedQuotes.length)];
  const authorLine = q.author ? `\n>\n> — **${q.author}**` : '';
  dv.paragraph(`> [!quote] ${q.text}${authorLine}`);
} else {
  dv.paragraph("> [!quote] 所有伟大的交易者都是从记录开始的。继续记录，持续进步。");
}
```
````

## 使用步骤

1. **复制** 上面的代码块
2. **打开** 目标模板（如交易日志模板(v1.0)）
3. **粘贴** 到文档最顶部（标题 `#` 之前）
4. **保存** — 打开阅读/预览模式即可看到效果

## 自定义

### 添加新的标签映射

在 `tagToCategory` 对象中新增一行：

```javascript
"你的标签名": "对应的类别名",
```

### 添加新的语录

直接编辑 [[09_知识库管理/引用数据库]]，按格式新增一行即可，无需修改模板。

### 改变显示样式

修改 `dv.span(...)` 中的 callout 类型，例如把 `quote` 换成 `tip`、`info` 等。
