## 📋 交易记录索引

```dataview
TABLE 
  date as "日期",
  trade_count as "交易次数",
  win_rate as "胜率",
  pnl as "盈亏"
FROM "03_财富投资/02_交易执行/01_交易日志"
WHERE contains(tags, "交易日志") AND file.name != "交易日志模板"
SORT date DESC
```

> [!tip] Dataview 提示
> 以上查询使用了 Dataview 插件。如果未安装，请在社区插件市场搜索 "Dataview" 安装启用。
