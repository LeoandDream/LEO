# LEO

这是我的个人知识库，用来管理生活、成长、事业、投资、工具与一切日常积累。

## 顶层结构

```text
.
├─ 00_Inbox/          # 收件箱 — 临时存放，待处理
├─ 01_个人成长/        # 阅读、技能、Self-Improvement
├─ 02_专业事业/        # 工作、项目、履历、面试
├─ 03_财富投资/        # 资产总览、交易日志、复盘与规则
│   └─ 02_交易执行/
│       ├─ 01_交易日志/    # 单笔交易记录（第三层）
│       ├─ 02_周期复盘/    # 周、月总结模板
│       └─ 03_交易规则/    # 纪律清单与资金管理
├─ 04_工具系统/        # Obsidian 配置、软件、工作流
├─ 05_生活健康/        # 健身、饮食、旅行、习惯
├─ 06_认知思维/        # 思维模型、决策框架、反直觉
├─ 07_资源模板/        # 复用模板、表格、检查清单
├─ 08_归档/            # 已完成/暂不沿用的内容
└─ .obsidian/          # Obsidian 配置（不含 workspace）
```

## 使用方式

1. 用 Obsidian 打开这个仓库。
2. 从 [[首页]] 开始导航。
3. 新内容先丢进 `00_Inbox`，再移动到对应目录。
4. 交易日志位于 `03_财富投资/02_交易执行/01_交易日志`。

## 设计原则

- 先记录，再整理，不要求一开始就完美
- 优先沉淀能复用、能复盘的信息
- 模板尽量简单，保证长期可坚持
- 仓库用于长期有效内容，不包含本地界面状态

## 同步说明

使用 GitHub 同步。`.obsidian/workspace.json` 等本地状态不会被跟踪，以减少无意义冲突。
