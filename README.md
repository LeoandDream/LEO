# LEO

个人知识库，用来管理生活、成长、事业、财富、工具与一切日常积累。

## 顶层结构

```text
.
├─ 00_Inbox/          # 收件箱 — 临时存放，待分类
├─ 01_个人成长/        # 个人成长 — 目标、习惯、学习路径
├─ 02_专业事业/        # 专业事业 — 工作、项目、履历
├─ 03_财富投资/        # 财富投资 — 资产、交易、复盘
│   ├─ 01_财富总览/        # 账户与资产总览
│   └─ 02_交易执行/        # 交易模块
│       ├─ 模板/           # 交易日志/周/月总结模板
│       ├─ 01_交易日志/    # 单笔交易记录
│       ├─ 02_周期复盘/    # 周/月复盘笔记
│       └─ 03_交易规则/    # 纪律与资金管理
├─ 04_工具系统/        # 工具系统 — 软件、自动化、工作流
├─ 05_生活健康/        # 生活健康 — 作息、运动、饮食
├─ 06_认知思维/        # 认知思维 — 模型、原则、方法论
├─ 07_资源模板/        # 资源模板 — 复用模板、清单
├─ 08_归档/            # 归档 — 已完成内容
├─ 09_知识库管理/      # 知识库自身管理
├─ pictrues/           # 图片素材
└─ .obsidian/          # Obsidian 配置（不含 workspace）
```

## 使用方式

1. 用 Obsidian 打开这个仓库。
2. 从 [[首页]] 开始导航。
3. 新内容先丢进 `00_Inbox`，整理后再移动到对应方向目录。
4. 完整架构说明见 [[09_知识库管理/LEO知识库构建思路]]。

## 设计原则

- 先记录，再整理，不要求一开始就完美
- 优先沉淀能复用、能复盘的信息
- 模板尽量简洁，保证长期可坚持
- 按人生方向分域，每个域按需决定内部层级
- 尚未开发的方向留空即可，不过度规划

## 同步说明

使用 GitHub 同步。`.obsidian/workspace.json` 等本地状态不会被跟踪，以减少无意义冲突。
