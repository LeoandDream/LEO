# LEO

这是我的个人知识库，用来管理生活、学习、交易以及长期积累的想法与记录。

目前它首先承载交易记录与复盘，之后会逐步扩展为一个更完整的个人管理系统。

## 这个仓库用来放什么

- 生活记录与待整理事项
- 学习笔记与长期知识积累
- 交易日志、周总结、月总结
- 规则清单、复盘结论与行动经验

## 当前结构

```text
.
├─ 交易日志/
│  ├─ 交易日志模板.md
│  ├─ 交易规则清单.md
│  ├─ 周总结模板.md
│  └─ 月总结模板.md
├─ 交易日志.md
└─ .obsidian/
```

## 使用方式

1. 用 Obsidian 打开这个仓库。
2. 从 `交易日志/交易日志模板.md` 开始记录单次或单日交易。
3. 按周、按月使用总结模板复盘。
4. 在首页 `交易日志.md` 汇总与回看。

## 设计原则

- 先记录，再整理，不要求一开始就完美
- 优先保留能复盘、能改进、能复用的信息
- 模板尽量简单，保证能长期坚持
- 仓库用于沉淀长期有效的内容，而不是临时界面状态
